# Телефонный справочник

## Основное меню:
1. Новая запись
2. Поиск в справочник


**Под меню  для новая запись:**
1. Создать контакт
2. Изменить контакт


**Под меню поиска:**
1. Поиск по фамилии
2. Поиск по комментарию
3. и тд


**Введенный текст принимается в словарь типа:**\
_{1:  
    Имя: Вася, \
Фамилия: Пупкин, \
контакт: {домашний:12345, рабочий: 4321}, \
город: \
примечание: абырвалг},\
2: \
    Имя: Костя, \
Фамилия: Жуков, \
контакт: {домашний:12345, рабочий: 4321}, \
город: \
примечание: абырвалг},
}_
